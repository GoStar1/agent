# 二次函数基础知识

## 定义

二次函数是形如 f(x) = ax² + bx + c 的函数，其中 a、b、c 是常数，且 a ≠ 0。

## 基本性质

1. **图像**: 二次函数的图像是一条抛物线
2. **开口方向**:
   - 当 a > 0 时，抛物线开口向上
   - 当 a < 0 时，抛物线开口向下

3. **顶点**: 抛物线的顶点坐标为 (-b/2a, (4ac-b²)/4a)

4. **对称轴**: x = -b/2a

## 常见形式

### 一般式
f(x) = ax² + bx + c

### 顶点式
f(x) = a(x - h)² + k
其中 (h, k) 是顶点坐标

### 交点式
f(x) = a(x - x₁)(x - x₂)
其中 x₁ 和 x₂ 是函数与 x 轴的交点

## 应用

二次函数在物理学中有广泛应用，例如：
- 抛体运动
- 自由落体运动
- 最优化问题

## 例题

**例1**: 求函数 f(x) = x² - 4x + 3 的顶点坐标。

**解**:
- a = 1, b = -4, c = 3
- 顶点横坐标: x = -b/2a = 4/2 = 2
- 顶点纵坐标: y = f(2) = 4 - 8 + 3 = -1
- 顶点坐标为 (2, -1)

**例2**: 求函数 f(x) = -2x² + 8x - 6 的最大值。

**解**:
- 因为 a = -2 < 0，抛物线开口向下，函数有最大值
- 最大值在顶点处取得
- x = -b/2a = -8/(-4) = 2
- 最大值 = f(2) = -8 + 16 - 6 = 2
